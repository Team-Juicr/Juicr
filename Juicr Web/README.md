# Juicr Web

Juicr Web is the browser companion surface for Juicr. It keeps familiar Home, Discovery, Library, and Settings areas available in a lightweight web experience.

## Highlights

- Browser-friendly Juicr layout for quick browsing and library checks.
- Local-first app shell with no required build step for basic static hosting.
- Optional setup paths stay user-controlled.
- Support information is kept limited to safe, non-private signals.

## Development

Open the static files directly or serve this folder with a local web server when testing browser behavior. Keep Web updates focused on companion browsing, clear navigation, and safe public-facing copy.
